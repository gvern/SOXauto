# Archive Directory

This directory contains archived test files that are no longer part of the active test suite.

## Archived Tests

- **`test_temporal_setup.py`**: Tests for the deprecated Temporal.io implementation
- **`test_retry_policy.py`**: Tests for the Temporal RetryPolicy configuration

These tests were archived when the project pivoted from Temporal.io to n8n for workflow orchestration.

> ⚠️ **Note**: These tests are not executed as part of the CI/CD pipeline.
